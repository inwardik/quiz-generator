# Quiz Generator - Enhanced Version

Улучшенный генератор интерактивных HTML quiz из Markdown файлов.

## Возможности

### Основные улучшения

- **Standalone HTML** - все стили и скрипты встроены в один файл
- **Современный дизайн** - красивый адаптивный интерфейс
- **Темная тема** - переключение между светлой и темной темой
- **Прогресс-бар** - визуальное отображение прогресса прохождения теста
- **Подсчет баллов** - в реальном времени
- **Финальные результаты** - детальная статистика по завершению
- **Мобильная адаптация** - корректное отображение на всех устройствах
- **Без зависимостей** - не требуется Bootstrap или другие библиотеки
- **Приватность** - данные не отправляются на сервер

### Функции квиза

- Перемешивание вопросов (опционально)
- Перемешивание ответов (опционально)
- Мгновенная обратная связь по ответам
- Показ правильных ответов при ошибке
- Возможность повторного прохождения
- Сохранение предпочтений темы в localStorage

## Требования

Python 3.6 или выше. Никаких внешних зависимостей!

## Установка

Просто скопируйте файл `to_html.py` в нужную директорию.

## Использование

### Базовое использование

```bash
python to_html.py quiz.md
```

Это создаст файл `quiz.html` в той же директории.

### Опции командной строки

```bash
# Указать имя выходного файла
python to_html.py quiz.md -o my-quiz.html

# Отключить перемешивание вопросов
python to_html.py quiz.md --no-shuffle-questions

# Отключить перемешивание ответов
python to_html.py quiz.md --no-shuffle-answers

# Включить темную тему по умолчанию
python to_html.py quiz.md --dark-mode

# Комбинация опций
python to_html.py quiz.md -o output.html --no-shuffle-questions --dark-mode
```

### Справка

```bash
python to_html.py --help
```

## Формат quiz.md

### Структура файла

```markdown
## Название теста

#### Текст вопроса 1

- [ ] Неправильный ответ 1
- [x] Правильный ответ
- [ ] Неправильный ответ 2
- [ ] Неправильный ответ 3

#### Текст вопроса 2

- [ ] Неправильный ответ
- [x] Правильный ответ 1
- [x] Правильный ответ 2
- [ ] Неправильный ответ
```

### Правила форматирования

1. **Заголовок теста** - используйте `## Название` (h2) в начале файла
2. **Вопросы** - начинаются с `####` (h4) или `###` (h3)
3. **Ответы** - список с чекбоксами:
   - `- [ ]` - неправильный ответ
   - `- [x]` или `- [X]` - правильный ответ
4. **Множественные правильные ответы** - поддерживаются, используйте несколько `[x]`

### Пример

См. файл `quiz.md` в этой директории для полного примера с 15 вопросами по Python.

## Особенности реализации

### Технические детали

- **Парсинг** - регулярные выражения для надежного парсинга Markdown
- **HTML/CSS/JS** - все встроено в один файл для портативности
- **Адаптивный дизайн** - CSS Grid и Flexbox
- **Темная тема** - CSS переменные для легкого переключения
- **LocalStorage** - сохранение настроек темы
- **Валидация** - проверка структуры quiz перед генерацией

### Архитектура кода

- `QuizParser` - класс для парсинга Markdown
- `HTMLGenerator` - класс для генерации HTML
- `main()` - обработка аргументов командной строки
